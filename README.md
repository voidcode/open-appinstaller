#OpenApp-Installer

This is a app-installer. Build on Ubuntu 14.04

It can install all apps defined in applist.json file.

* Filezilla
* Termainator
* Gimp
* Inkscape
* Shutter
* Wireshark
* Git
* Mysql Workbench

You can now seleted or unselected any of then before beginning the installed.

#Run it
´´´
    sudo chmod +x run
    ./run
´´´
